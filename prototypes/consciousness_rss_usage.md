## Troubleshooting

### No Items Found
- Try `sync` mode instead of `discover` to see all processed items
- Check internet connectivity for RSS feed access
- Original cfrss.py may have restrictive 0.2 threshold - use enhanced version

### Low Relevance Scores
- Real-world content typically scores 0.05-0.15 (not 0.6 like test content)
- This is normal - the system is calibrated for actual RSS feed patterns
- Enhanced thresholds account for this reality

### Adding New Feeds
- Edit the `default_feeds` dictionary in enhanced_cfrss.py
- Add consciousness-specific feed categories as needed
- Restart to pick up new feeds
